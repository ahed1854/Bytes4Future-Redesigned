# Bytes4future Website Redesign

A modern redesign of the Bytes4future website, built with simple HTML, CSS, and JavaScript. This project focuses on a clean, responsive layout and improved user experience.

- HTML5
- CSS3 (Flexbox/Grid for layout)
- JavaScript (ES6+)
